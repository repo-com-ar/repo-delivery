# repo-delivery

APP PWA Mobile First destinada a ser utilizada por los repartidores.

Dado que los repartidores siempre la utilizarán desde su celular, la aplicación está diseñada con un enfoque Mobile First, optimizada para pantallas pequeñas y uso en movimiento.

## Funcionalidades principales

- **Recepción de pedidos**: Los repartidores reciben en tiempo real los pedidos que tienen asignados para entregar.
- **Confirmación de entrega**: Permite marcar cada pedido como entregado una vez completada la entrega.
- **Mapas y navegación**: Integración de mapas para visualizar la ubicación exacta donde debe realizarse cada entrega.

## Tecnología

Progressive Web App (PWA) — instalable en el celular del repartidor sin necesidad de pasar por una tienda de aplicaciones, con soporte offline y notificaciones push.
